# Ejércitos

Lenguaje utilizado: Smalltalk.

## Consideraciones
- Las unidades aumentan sus años de vida después de entrenar o transformarse. El enunciado no lo aclara.
- No se guardan las estadísticas de los ejercitos en el historial (el estado de sus unidades, puntos de fuerza o monedas).
    El enunciado no lo pide
- En caso de empate en una batalla cada ejército pierde la unidad de mayor puntaje.
- Se espera que en un futuro haya más tipos de unidades.
